# LP-0012: Learning Path - Smart Contract Developer

## Abstract

**"I want to build apps, protocols, and money."**

This path takes you from EVM basics to production DeFi on Lux.

## Audience

Solidity developers, Web3 teams, protocol engineers.

## Prerequisites

Complete [LP-0011: Core Protocol Path](/docs/lp-0011-learning-path-core/) first.

## Outcome

After completing this path, you will be able to:
- Deploy smart contracts on C-Chain
- Build token standards (LRC-20/721/1155)
- Implement DeFi primitives (AMMs, lending, vaults)
- Optimize for Lux precompiles
