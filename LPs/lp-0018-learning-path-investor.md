# LP-0018: Learning Path - Investor & Institutional

## Abstract

**"Where does value accrue and why?"**

This path provides capital deployment understanding without deep technical requirements.

## Audience

## Prerequisites

None. This path includes Core Protocol essentials.

## Outcome

After completing this path, you will understand:
- Token economics and value flows
- Platform revenue model
- DeFi surface area
- Trading liquidity
- Governance structure
- ESG commitments
